# Proyecto 1: Generador de números aleatorios

Proyecto 1 para la clase de métodos cuentitativos y simulación. 

## Descripción General

### Generación
Para la generación de números aleatorios se utilizaron los siguientes métodos para la generación:
1. Método de los Centros Cuadrados
2. Método Congruencial 
3. Método Congruencial Mixto con validación del Teorema de HULL-DOBELL
4. Generador Multiplicativo 
5. Método Congruencial Lineal

### Validación
1. Prueba de Chi-Cuadrada
2. Prueba de Kolmogorov-Smirnov
